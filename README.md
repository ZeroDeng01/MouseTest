# 鼠标测试工具

一款简单易用的鼠标测试工具，可以测试鼠标的按键、滚轮是否出现点击问题，可以评估鼠标是否有粘连问题。

**技术框架**：.NET8 WinForms

**测试兼容**：`Win10`/`Win11`+`MySQL 8.0`
